# ZACO Robot Vacuum

Home Assistant integration for ZACO robot vacuums (A10 and other models built on the
iRobotics/3irobotix platform). It talks to the Aliyun IoT Living Platform cloud — the same
backend used by the official ZACOHome app — so no local rooting or firmware modification is needed.

**Features:** vacuum entity with start/stop/pause/return/locate, live floor map camera with robot
path, room and zone cleaning, go-to-point, suction/water controls, and consumable sensors.

Log in with your ZACOHome app email and password during setup; the integration discovers your
devices automatically. See the [README](https://github.com/eXaminator/ha-zaco-vacuum) for the full
entity list and map-card setup.
